TabBar 图标说明
================

微信小程序 tabBar 需要以下图标文件：

1. home.png - 首页图标（未选中）
2. home-active.png - 首页图标（选中）
3. list.png - 错题列表图标（未选中）
4. list-active.png - 错题列表图标（选中）
5. search.png - 搜索图标（未选中）
6. search-active.png - 搜索图标（选中）
7. add.png - 添加图标（未选中）
8. add-active.png - 添加图标（选中）

图标要求：
- 尺寸：81px × 81px
- 格式：PNG
- 未选中颜色：#999999
- 选中颜色：#1890ff

可以从以下图标库获取：
- 阿里巴巴矢量图标库：https://www.iconfont.cn/
- IconPark：https://iconpark.oceanengine.com/

下载图标后放入此目录即可。
