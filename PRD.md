# 学生错题收录系统 - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品名称
错题本 (MistakeBook)

### 1.2 产品定位
面向学生的智能错题收录与管理工具，帮助学生高效记录、整理、复习错题，提升学习效率。

### 1.3 核心价值
- **便捷记录**：支持文本和图片两种录入方式
- **智能识别**：OCR识别图片中的题目内容
- **系统管理**：按学科、知识点、时间等维度分类管理
- **复习提醒**：基于艾宾浩斯记忆曲线的智能复习计划

---

## 二、核心功能模块

### 2.1 错题录入（一期核心）

#### 功能1：文本录入
- 支持手动输入题目内容
- 支持手动输入正确答案
- 支持手动输入错误原因分析
- 支持选择学科（数学、物理、化学、生物等）
- 支持添加标签/知识点
- 支持Markdown格式排版

#### 功能2：图片上传
- 支持拖拽上传图片
- 支持多张图片上传（最多10张）
- 支持常见图片格式（JPG, PNG, WEBP, GIF）
- 单张图片限制：10MB
- 自动压缩优化图片大小
- 图片预览和旋转功能

#### 功能3：OCR识别（二期规划）
- 自动识别图片中的题目文本
- 支持手写体识别（对接AI大模型）
- 识别结果可编辑确认
- 识别度目标：印刷体>95%，手写体>85%

### 2.2 错题管理

#### 列表展示
- 分页展示错题列表
- 支持按学科筛选
- 支持按标签筛选
- 支持按时间范围筛选
- 支持关键词搜索
- 支持按难度/掌握程度排序

#### 错题详情
- 完整展示题目、正确答案、错误分析
- 图片放大查看
- 编辑/删除功能
- 标记掌握状态（未掌握/半掌握/已掌握）

#### 分类管理
- 学科分类：数学、物理、化学、生物、英语等
- 自定义标签：用户可创建和管理标签
- 知识点归类：支持知识点层级管理

### 2.3 错题复习

#### 复习计划
- 基于艾宾浩斯遗忘曲线生成复习提醒
- 支持手动触发复习
- 复习模式：遮罩答案模式（先看题目，再显示答案）

#### 复习统计
- 每日复习数量统计
- 掌握进度跟踪
- 各学科复习完成率

### 2.4 数据导出

#### 导出格式
- 支持导出为PDF文档
- 支持导出为Word文档
- 支持导出为Excel表格
- 支持按学科/时间/标签批量导出

### 2.5 用户系统（二期规划）

#### 用户管理
- 学生账号注册/登录
- 个人资料管理
- 学习数据同步

#### 数据同步
- 多设备数据同步
- 云端自动备份

## 三、技术架构

### 3.1 技术选型

#### 前端
- **框架**：React 18 + TypeScript
- **UI组件库**：Ant Design
- **状态管理**：Zustand
- **HTTP客户端**：Axios
- **构建工具**：Vite
- **富文本编辑器**：Tiptap (可选) 或基础textarea

#### 后端
- **语言**：Java 23
- **框架**：Spring Boot 3.2.x
- **ORM**：Spring Data JPA + Hibernate
- **数据库**：H2(开发) / MySQL 8.0(生产)
- **API文档**：Knife4j (Swagger)
- **图片处理**：Thumbnailator
- **OCR**：Spring AI / Tesseract(可选) / 云服务API

#### 部署
- **容器化**：Docker
- **数据库**：MySQL + 可选Redis缓存

## 四、数据库设计

### 4.1 ER关系图

```
用户(User) 1 —— * 错题(Mistake)
错题(Mistake) * —— * 标签(Tag) [多对多]
错题(Mistake) 1 —— * 错题图片(MistakeImage)
```

### 4.2 核心表结构

#### 用户表 (t_user)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| username | VARCHAR(50) | 用户名 |
| password | VARCHAR(100) | 密码(加密) |
| email | VARCHAR(100) | 邮箱 |
| avatar | VARCHAR(255) | 头像URL |
| status | TINYINT | 状态 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

#### 错题表 (t_mistake)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID |
| title | VARCHAR(200) | 题目摘要 |
| content | TEXT | 题目内容 |
| correct_answer | TEXT | 正确答案 |
| wrong_reason | TEXT | 错误分析 |
| subject | VARCHAR(20) | 学科 |
| difficulty | TINYINT | 难度(1-5) |
| tags | VARCHAR(500) | 标签JSON |
| status | TINYINT | 状态(0未掌握 1半掌握 2已掌握) |
| review_count | INT | 复习次数 |
| next_review_time | DATETIME | 下次复习时间 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

#### 错题图片表 (t_mistake_image)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| mistake_id | BIGINT | 错题ID |
| image_url | VARCHAR(255) | 图片路径 |
| image_type | TINYINT | 类型(1题目 2答案 3解析) |
| sort_order | INT | 排序 |
| create_time | DATETIME | 创建时间 |

#### 标签表 (t_tag)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID |
| name | VARCHAR(50) | 标签名称 |
| color | VARCHAR(7) | 颜色 |
| create_time | DATETIME | 创建时间 |

## 五、API接口设计

### 5.1 统一响应格式

```json
{
    "code": 200,
    "message": "success",
    "data": {},
    "timestamp": 1700000000000
}
```

### 5.2 错题接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建错题 | POST | /api/v1/mistakes | 创建新的错题记录 |
| 获取列表 | GET | /api/v1/mistakes | 分页获取错题列表 |
| 获取详情 | GET | /api/v1/mistakes/{id} | 获取错题详情 |
| 更新错题 | PUT | /api/v1/mistakes/{id} | 更新错题信息 |
| 删除错题 | DELETE | /api/v1/mistakes/{id} | 删除错题 |
| 搜索错题 | GET | /api/v1/mistakes/search | 搜索错题 |
| 更新状态 | PUT | /api/v1/mistakes/{id}/status | 更新掌握状态 |
| 导出错题 | GET | /api/v1/mistakes/export | 导出错题 |

### 5.3 图片接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 上传图片 | POST | /api/v1/images/upload | 上传图片 |
| 获取图片 | GET | /api/v1/images/{id} | 获取图片 |
| 删除图片 | DELETE | /api/v1/images/{id} | 删除图片 |

### 5.4 标签接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建标签 | POST | /api/v1/tags | 创建标签 |
| 获取列表 | GET | /api/v1/tags | 获取标签列表 |
| 删除标签 | DELETE | /api/v1/tags/{id} | 删除标签 |

## 六、非功能性需求

### 6.1 性能要求
- API响应时间 < 500ms
- 图片上传 < 5s
- 列表分页加载 < 1s

### 6.2 安全要求
- 密码加密存储
- 图片防盗链
- API防刷限流
- SQL注入防护

### 6.3 兼容性
- 浏览器：Chrome/Edge/Safari/Firefox最新版本
- 响应式设计（可选）
