第一单元 · 多彩的春天

1. 古诗三首 绝句 / 惠崇春江晚景 / 三衢道中
  • 《绝句》杜甫（唐）："迟日江山丽，春风花草香。泥融飞燕子，沙暖睡鸳鸯。"——春天生机勃勃
  • 《惠崇春江晚景》苏轼（宋）："竹外桃花三两枝，春江水暖鸭先知。蒌蒿满地芦芽短，正是河豚欲上时。"——早春景色
  • 《三衢道中》曾几（宋）："梅子黄时日日晴，小溪泛尽却山行。绿阴不减来时路，添得黄鹂四五声。"——初夏山行
  • 易错字：鸳鸯、蒌蒿、芦芽、豚、衢
2. 燕子 郑振铎
  • 主题：描写燕子的外形、飞行、停歇，表达对春天和燕子的喜爱
  • 重点词语：乌黑光亮、俊俏轻快、活泼机灵、光彩夺目、聚拢、增添、偶尔、荡漾
  • 修辞：比喻（"一身乌黑光亮的羽毛，一对俊俏轻快的翅膀，加上剪刀似的尾巴"）
  • 写法：按"外形→飞行→停歇"的顺序观察描写
3. 荷花 叶圣陶
  • 主题：描写荷花的各种姿态，感受大自然的美丽
  • 重点词语：挨挨挤挤、翩翩起舞、饱胀、破裂、仿佛、衣裳
  • 修辞：比喻（"荷叶挨挨挤挤的，像一个个碧绿的大圆盘"）；排比（"有的……有的……有的……"）
  • 重点句："我忽然觉得自己仿佛就是一朵荷花"——想象手法，人花合一
4. 昆虫备忘录 汪曾祺
  • 主题：介绍蜻蜓、瓢虫、独角仙、蚂蚱等昆虫的特点
  • 重点词语：备忘录、复眼、琢磨、款款、严丝合缝
  • 写法：观察笔记式写法，语言生动有趣

第二单元 · 寓言故事

5. 守株待兔 寓言·文言文
  • 原文：宋人有耕者。田中有株。兔走触株，折颈而死。因释其耒而守株，冀复得兔。兔不可复得，而身为宋国笑。
  • 寓意：不能把偶然的事当成必然，不能心存侥幸，不劳而获
  • 重点字：株（树桩）、走（跑）、触（撞）、因（于是）、释（放下）、耒（农具）、冀（希望）
6. 陶罐和铁罐 黄瑞云
  • 主题：铁罐看不起陶罐，多年后铁罐锈蚀而陶罐完好——每个人都有长处和短处
  • 重点词语：骄傲、傲慢、谦虚、懦弱、轻蔑、恼怒、相提并论、和睦相处
  • 道理：要善于看到别人的长处，正视自己的短处
7. 鹿角和鹿腿 伊索寓言
  • 主题：鹿嫌腿丑却靠腿逃生，赞角美却被角挂住——事物各有价值
  • 重点词语：倒映、匀称、抱怨、机灵、犹豫、灰心丧气
  • 道理：物各有长短，不能只看外表
8. 池子与河流 克雷洛夫寓言·诗歌
  • 主题：池子贪图安逸最终干涸，河流奔流不息长流不断
  • 道理：不要贪图安逸，要积极进取，才能保持生命力

第三单元 · 传统文化

9. 古诗三首 元日 / 清明 / 九月九日忆山东兄弟
  • 《元日》王安石："爆竹声中一岁除，春风送暖入屠苏。千门万户曈曈日，总把新桃换旧符。"——春节景象
  • 《清明》杜牧："清明时节雨纷纷，路上行人欲断魂。借问酒家何处有？牧童遥指杏花村。"——清明节
  • 《九月九日忆山东兄弟》王维："独在异乡为异客，每逢佳节倍思亲。遥知兄弟登高处，遍插茱萸少一人。"——重阳思亲
10. 纸的发明
  • 主题：介绍造纸术的发明过程，感受中国古人的智慧
  • 重点词语：创造、携带、保存、普及、传承、粗糙、改进
  • 说明顺序：按时间顺序（西汉→东汉→后来）介绍纸的演变
11. 赵州桥
  • 主题：介绍赵州桥的雄伟、坚固、美观，赞美劳动人民的智慧
  • 重点词语：创举、坚固、雕刻、遥望、智慧、遗产
  • 说明方法：列数字（"桥长五十多米，有九米多宽"）
12. 一幅名扬中外的画
  • 主题：介绍《清明上河图》的内容和价值
  • 重点词语：名扬中外、传神、形态各异、来来往往、热闹非凡

第四单元 · 观察与发现

13. 花钟
  • 主题：不同的花在不同的时间开放，可以组成"花钟"
  • 重点词语：争奇斗艳、芬芳迷人、留心、观察、绽开、苏醒
  • 写法：用不同的说法表达鲜花开放（拟人手法）
14. 蜜蜂 法布尔
  • 主题：法布尔做实验验证蜜蜂有辨认方向的能力
  • 重点词语：辨认、大概、阻力、推测、包括、准确无误
  • 科学精神：严谨的实验态度——"听说……我想……我做了实验……"
15. 小虾
  • 主题：描写小虾的外形和生活习性，表达对小虾的喜爱
  • 重点词语：空隙、追逐、搏斗、通体透明、一翘一翘

第五单元 · 想象的世界

16. 宇宙的另一边 陈诗哥
  • 主题：一个孩子想象宇宙另一边的世界——奇妙的想象
  • 重点词语：浩瀚、流淌、秘密、思绪、一篇
  • 写法：反向想象（这边下雨→那边晴天；这边冬→那边夏）
17. 我变成了一棵树 顾鹰
  • 主题："我"不想吃饭变成了一棵树，发生了很多有趣的事
  • 重点词语：希望、形状、狐狸、继续、麻烦、秘密
  • 想象手法：大胆而合理的想象，从现实出发的奇思妙想

第六单元 · 多彩的童年

18. 童年的水墨画 张继楼·诗歌
  • 主题：三首小诗描绘溪边钓鱼、江上戏水、林中采蘑菇的童年画面
  • 重点词语：水墨画、垂柳、梳妆、戏耍、清清爽爽
  • 写法：用诗意的语言捕捉童年生活的美好瞬间
19. 剃头大师 秦文君
  • 主题："我"给小沙剃头的趣事——童年的天真与乐趣
  • 重点词语：剃头、师傅、冤家、规规矩矩、耿耿于怀
  • 语言特色：幽默风趣，充满童真
20. 肥皂泡 冰心
  • 主题：吹肥皂泡的过程和美丽的想象
  • 重点词语：肥皂泡、廊子、脆薄、颤巍巍、光影零乱
  • 写法：按"做肥皂水→吹泡泡→看泡泡→想泡泡"的顺序
21. 我不能失信
  • 主题：宋庆龄小时候守信的故事——诚实守信
  • 重点词语：失信、道歉、解释、扑空、惦记
  • 道理：做人要守信用，说到做到

第七单元 · 大自然的奥秘

22. 我们奇妙的世界 彼得·西摩
  • 主题：从天空和大地两方面展现世界的奇妙与美丽
  • 重点词语：奇妙、呈现、变幻、诱人、圆润、感叹
  • 写法：按"天空→大地"的结构，首尾呼应
23. 海底世界
  • 主题：海底景色奇异、物产丰富
  • 重点词语：波涛澎湃、窃窃私语、蕴藏、丰富、差异
  • 写法：从声音、动物、植物、矿藏等多方面介绍海底世界
24. 火烧云 萧红

  • 重点词语：紫檀色、笑盈盈、红彤彤、恍恍惚惚、凶猛
  • 写法：颜色变化（红彤彤→金灿灿→半紫半黄……）；形状变化（马→狗→狮子）

第八单元 · 有趣的故事

25. 慢性子裁缝和急性子顾客 周锐
  • 主题：慢性子裁缝和急性子顾客之间的有趣对话
  • 重点词语：性子、一卷、交货、取衣服、负责、恼怒
  • 写法：通过对话推动情节，对比鲜明
26. 方帽子店
  • 主题：人们一直做方帽子，后来出现了各种各样的新帽子——不能因循守旧
  • 道理：要勇于创新，不能墨守成规
27. 漏 民间故事
  • 主题：老虎和贼都想偷驴，却被"漏"吓跑了——误会产生的喜剧
  • 重点词语：翻山越岭、走南闯北、倒栽葱、旋风

重点句式训练

1. 比喻句

  • 荷叶挨挨挤挤的，像一个个碧绿的大圆盘。
  • 一身乌黑光亮的羽毛，一对俊俏轻快的翅膀，加上剪刀似的尾巴。
  • 仿写：弯弯的月亮像________________。

2. 拟人句

  • 凌晨四点，牵牛花吹起了紫色的小喇叭。
  • 不光是我一朵，一池的荷花都在舞蹈。
  • 仿写：小鸟在枝头________________。

3. 排比句

  • 有的才展开两三片花瓣儿。有的花瓣儿全展开了。有的还是花骨朵儿。
  • 仿写：下课了，同学们有的________，有的________，有的________。

4. 反问句 ↔ 陈述句

  • 反问：这不恰好说明他做对了吗？ → 陈述：这恰好说明他做对了。
  • 练习：难道我们不应该保护环境吗？ → ________________________

5. "把"字句 ↔ "被"字句

  • 把字句：洪水把小村庄吞没了。 → 被字句：小村庄被洪水吞没了。

6. 关联词

  • 因果：因为……所以…… / 之所以……是因为……
  • 转折：虽然……但是…… / 尽管……还是……
  • 条件：只要……就…… / 只有……才……
  • 假设：如果……就…… / 即使……也……

阅读理解与写作要点

阅读理解方法

  • 找中心句：关键句通常在段首或段尾
  • 理解词语：联系上下文理解词语的含义
  • 概括段意：谁 + 在什么情况下 + 做了什么 + 结果怎样
  • 体会感情：抓住关键词句，体会作者表达的情感
  • 说明方法识别：列数字、作比较、打比方、举例子

本学期常见作文题

📝 写景类

题目：这儿真美 / 我喜欢的季节 / 校园一角
按一定顺序观察描写；用上比喻、拟人等修辞；调动多种感官（看、听、闻、摸）；融入自己的感受。

📝 写事类

题目：一件___的事 / 那次玩得真高兴 / 这件事真有趣
把事情的起因、经过、结果写清楚；重点部分要详细描写（动作、语言、心理）；写出自己的感受。

📝 写人类

题目：我的___ / 我身边有特点的人 / 猜猜他是谁
围绕人物的一个特点来写；通过具体事例来表现；抓住外貌、语言、动作、心理描写。

📝 想象类

题目：这样想象真有趣 / 假如我会变 / 我变成了___
大胆想象，合情合理；故事要有起因、经过、结果；想象要围绕一个中心展开。
