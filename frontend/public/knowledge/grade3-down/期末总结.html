核心语法要点

1. be动词 (am / is / are)

  • I am ... (我是...)
  • He / She / It is ... (他/她/它是...)
  • You / We / They are ... (你/我们/他们是...)
I am a student. / She is my sister. / They are happy.
我是一名学生。 / 她是我的姐姐。 / 他们很开心。

2. 一般现在时（第三人称单数）

主语为 he / she / it / 单数名词时，动词加 -s 或 -es：

  • like → likes   eat → eats   go → goes   watch → watches   have → has
He likes apples. / She watches TV every day. / The cat has a long tail.
他喜欢苹果。 / 她每天看电视。 / 这只猫有一条长尾巴。

3. 现在进行时 (be + doing)

表示正在发生的动作，结构：am/is/are + 动词ing

  • 一般加 -ing：play → playing, read → reading
  • 以e结尾去e加 -ing：make → making, write → writing
  • 双写末尾辅音加 -ing：run → running, swim → swimming
I am reading. / He is running. / They are playing.
我正在读书。 / 他正在跑步。 / 他们正在玩。

4. 疑问句

Do / Does 开头的一般疑问句：

  • Do you like apples? — Yes, I do. / No, I don't.
  • Does he like apples? — Yes, he does. / No, he doesn't.

Can 开头的情态动词疑问句：

  • Can you swim? — Yes, I can. / No, I can't.
  • Can she dance? — Yes, she can. / No, she can't.

特殊疑问句（Wh-questions）：

  • What is this? — It's a book. (这是什么？)
  • What colour is it? — It's red. (什么颜色？)
  • What time is it? — It's seven o'clock. (几点了？)
  • Where is the cat? — It's under the table. (在哪里？)
  • Who is he? — He is my brother. (他是谁？)
  • How old are you? — I'm nine. (你几岁？)
  • How many books? — Five books. (多少本书？)

5. 否定句

  • I am not ... / He is not (isn't) ...
  • I don't like ... / He doesn't like ...
  • I can't swim.

6. There is / There are

表示某处有某物：

  • There is a cat on the table. (桌上有一只猫。)
  • There are three books on the desk. (桌上有三本书。)

9. 可数名词复数变化规则

  • 一般加 -s：book → books, cat → cats
  • 以 s, x, ch, sh 结尾加 -es：bus → buses, box → boxes
  • 以辅音+y 结尾，变 y 为 i 加 -es：baby → babies, family → families
  • 以 f/fe 结尾，变 f/fe 为 v 加 -es：knife → knives, leaf → leaves
  • 不规则变化：child → children, foot → feet, tooth → teeth, man → men, woman → women, mouse → mice, fish → fish, sheep → sheep

易混淆知识点

⚠️ a / an 的用法

  • a book, a pen — 辅音音素开头
  • an apple, an egg, an orange — 元音音素开头

⚠️ this / that / these / those

  • this (这个) ↔ these (这些) — 近指
  • that (那个) ↔ those (那些) — 远指

⚠️ some / any

  • some 用于肯定句：I have some apples.
  • any 用于否定句和疑问句：Do you have any apples?

⚠️ too / also / either

  • too / also 用于肯定：I like apples too. / I also like apples.
  • either 用于否定：I don't like apples either.
